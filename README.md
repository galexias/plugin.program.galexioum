# Galexioum Wizard

Galexioum Wizard is a Kodi maintenance and utility addon providing
backup/restore, save data management and system cleanup tools.

## Features
- Backup & Restore (custom folder support)
- Save Data for Trakt & Real-Debrid (POV)
- Maintenance tools (cache, thumbnails, packages)
- Log viewer & uploader
- Clean and simplified UI
