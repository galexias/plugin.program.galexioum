# Galexioum Wizard

**Galexioum Wizard** is a Kodi maintenance and utility addon providing backup/restore, save data management and system cleanup tools, designed with a clean and simplified user interface.

---

## âœ¨ Features

- **Backup & Restore**
  - Custom backup folder selection
  - Full Kodi data backup & restore

- **Save Data**
  - Trakt save & restore
  - Real-Debrid save & restore (POV support)

- **Maintenance Tools**
  - Cache cleaning
  - Thumbnails cleanup
  - Packages cleanup

- **Logs**
  - Built-in log viewer
  - Log uploader for troubleshooting

- **UI**
  - Clean and simplified menu structure
  - Reorganized main & maintenance menus
